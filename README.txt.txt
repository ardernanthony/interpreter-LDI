README - Language Interpreter (Stage 4)

This interpreter supports:
- integer and float math
- string concatenation
- global variable assignment and usage
- basic type checking
- print statements
- line-by-line execution
- comment support (lines beginning with '#')

Usage:

Put 'test-code' in a file inside the `examples/` folder, then run. for example;

   python src/interpreter.py examples/test-code.txt

Examples provided:
- calc1.txt
- vars.txt
- floatTest.txt
- stringCatTest.txt
- errorTest.txt
